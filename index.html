Git & Github

Overview about Git & GitHub

IMPORTANT: Git & GitHub are not the same


Git is similar to "CMD Promt" on Windows which is a version control tool

Github is a git site hosting repository

How to download Git

Simply open a broswer and search for Git

Select either Git or Download - Git

Press Downloads and choose your operating system (MAC, Window, Linux) and click download

Basic comands about Git

There are 4 basic Git command that you will have to use everytime


1. Git Pull
To get the latest repository

2. Git Add - (git add index.html)
To add file(s) to the index

3. Git Commit - (git commit -m "a msg")
To commit change(s) that you have made

4. Git Push
To push file(s) you just change to the repository
